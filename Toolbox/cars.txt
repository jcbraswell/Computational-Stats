These data were extracted from a data set that is included in the MATLAB Statistics
Toolbox called carbig. The smaller data set given below includes two measurements
or predictor variables on small model cars: weight (x1) and horsepower (x2). The
response variable is miles per gallon.

Note that there is an error in the text (2nd Edition). We have weight listed as
x2 and horsepower as x3 in Appendix F. These are really x1 (weight) and x2
(horsepower).
